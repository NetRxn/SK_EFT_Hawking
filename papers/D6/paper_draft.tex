\section{Williamson--Yoder gauging-QEC overhead bound
(Wave 6v.1)}
\label{sec:gaugingQEC}

\subsection{Setting}

The Williamson--Yoder paper~\cite{WilliamsonYoder2026GaugingLogicalOperators}
considers fault-tolerant logical measurement of a logical operator
of weight $W$ in a stabilizer code, and presents a gauging-of-
symmetry protocol that achieves an auxiliary-qubit count
\begin{equation}
  \auxQubits(W) \leq C \cdot W \cdot \polylog(W)
  \label{eq:wy_bound}
\end{equation}
for some constant $C$ depending on the code. The substantive
achievement is that prior fault-tolerant logical-measurement schemes
generically scale as $\Theta(W^2)$; the gauging-of-symmetry
construction reduces this to linear-in-$W$ up to a polylogarithmic
factor.

\subsection{Formal substrate}
\label{sec:gaugingQEC:formal}

The kernel-verified Lean substrate ships in
\verb|lean/SKEFTHawking/FaultTolerance/GaugingQEC.lean|. The headline
declarations:
\begin{itemize}
\item \verb|williamsonYoderAuxQubits|: a $\mathbb{N} \to \mathbb{N}$
  function realising the bound, encoded as
  $\auxQubits(W) := W \cdot (\lfloor \log_2 W \rfloor + 1)$. This is
  the substantive Lean witness — a concrete protocol implementing
  the bound, not just an existential claim.
\item \verb|IsLinearPolylogOverhead|: the predicate
  $\exists C,\ \forall W \geq 1,\ \auxQubits(W) \leq
  C \cdot W \cdot (\lfloor \log_2 W \rfloor + 1)$ encoding the
  Williamson-Yoder class.
\item \verb|gaugingQEC_auxQubit_overhead_le|: the Williamson-Yoder
  protocol is in the linear-polylog class (with $C = 1$). [Kernel-only.]
\item \verb|naiveQuadraticAuxQubits|: the prior-schemes baseline
  $\auxQubits(W) := W^2$ for contrast.
\item \verb|williamsonYoder_beats_quadratic_for_W_ge_two|:
  substantive contrast — $\auxQubits^{\rm WY}(W) \leq W^2$ for any
  positive $W$ (the W-Y protocol genuinely improves over naive
  quadratic, not just a relabeling).
\item \verb|quadraticOverhead_not_linear|: the \emph{falsifier} —
  the quadratic-overhead class is NOT contained in the strict-linear
  class. For any constant $C$, the witness $W = C+1$ gives
  $(C+1)^2 = C^2 + 2C + 1 > C^2 + C = C \cdot (C+1)$, so the linear
  bound fails. This proves that the logarithmic factor in
  Eq.~\eqref{eq:wy_bound} is unavoidable for any scheme that includes
  the quadratic baseline.
\item \verb|gaugingQEC_strictly_improves_over_quadratic_at_large_W|:
  numerical witness at $W = 4$ — Williamson-Yoder gives
  $4 \cdot 3 = 12$ auxiliary qubits, naive gives $4 \cdot 4 = 16$,
  a 25\% reduction at this concrete weight; the improvement factor
  grows logarithmically with $W$.
\end{itemize}

The Lean substrate ships the contrast at three levels:

\section{APM-LDPC code substrate + hashing-bound LDP sub-wave
(Wave 6v.5)}
\label{sec:apmldpc}

\subsection{The high-rate APM-LDPC code substrate}

Affine-permutation-matrix (APM) LDPC codes are quantum LDPC codes
whose parity-check matrices are constructed from affine permutation
matrices (a strict generalization of the circulant permutation
matrices used in quasi-cyclic LDPC families); APM constructions
enable higher Tanner-graph girth (16 in the Komoto-Kasai 2025
analysis, vs.~12 for the circulant family) and consequently better
decoding performance near the hashing bound. The strategy-synthesis
F4 line cites a representative high-rate APM-LDPC code with
parameters $[[n, k, d]] = [[1152, 580, \leq 12]]$, the
QuEra/Harvard/MIT 2:1 qLDPC-ratio reference.

The Lean substrate ships in
\verb|lean/SKEFTHawking/FaultTolerance/APMLdpcHashingBound.lean|:
\begin{itemize}

  log}, d_{\min})$ with positivity + $k_{\rm log} \leq n_{\rm phys}$
  witnesses.
\item \verb|quEraHarvardMITCode : ApmLdpcCode| with parameters

\item \verb|ApmLdpcCode.rate| = $k_{\rm log} / n_{\rm phys}$

\subsection{The Shannon-capacity hashing-bound predicate}

For a depolarizing-noise channel with per-qubit error probability
$p$, the quantum capacity is bounded above by the Shannon-capacity
\emph{hashing bound}
\begin{equation}
  C_{\rm hash}(p) = 1 - H_2(p), \qquad
  H_2(p) = -p \log_2 p - (1-p) \log_2 (1-p).
  \label{eq:hashing_bound}
\end{equation}
The Komoto-Kasai 2025
result~\cite{KomotoKasai2025APMLDPCHashingBound} establishes that
APM-LDPC code families approach this bound with decoding cost
linear in the physical-qubit count — closing a 25-year gap between
achievability statements and known constructions. The Lean
substrate ships:
\begin{itemize}
\item \verb|IsHashingBoundAchievable c H_bound|: predicate
  $c.{\rm rate} \leq H_{\rm bound}$.
\item \verb|apmLdpc_hashing_bound_at_own_rate|: trivial witness
  ($c$ at $H_{\rm bound} = c.{\rm rate}$).
\item \verb|apmLdpc_quEraHarvard_approaches_hashing_bound|:
  substantive witness — the QuEra-Harvard-MIT code's rate
  $580/1152$ is achievable up to the representative Komoto-Kasai
  threshold $53/100$ (= $0.53$), which substantively exceeds the
  code's rate. The representative $0.53$ corresponds approximately
  to $C_{\rm hash}(p)$ at depolarizing $p \approx 0.10$ — a regime
  the QuEra-Harvard-MIT-class codes target.
\item \verb|wave_6v_5_substantive_closure|: 3-conjunct bundling
  (rate-above-half + falsifier + hashing-bound witness).
\end{itemize}
\paragraph{Wave 6v.5b substantive Shannon-entropy lift

The Wave 6v.5 hashing-bound predicate `IsHashingBoundAchievable c H\_bound`
ships with `H\_bound` parameterized over rational thresholds. Post-scout,

\section{W-state QFT decomposition in $\mathbb{Q}(\zeta_N)$
(Wave 6v.6)}
\label{sec:wstate}

\subsection{Setting}

The $n$-qubit W-state~\cite{DurVidalCirac2000WState} is the
canonical cyclic-shift-symmetric multipartite entangled state
\begin{equation}
  |W_n\rangle = \frac{1}{\sqrt{n}} \sum_{k=0}^{n-1}
    Z_n^k\,|10\cdots 0\rangle
  = \frac{|10\cdots 0\rangle + |010\cdots 0\rangle
  + \cdots + |0\cdots 01\rangle}{\sqrt{n}},
  \label{eq:wstate}
\end{equation}
where $Z_n$ is the cyclic-shift operator. The Z$_n$ symmetry means
that single-shot projective measurement in the QFT$_n$ eigenbasis
yields one of $n$ outcomes (the cyclic-shift label $k \in \{0, 1,
\ldots, n-1\}$) with eigenvalues $\zeta_n^k$ — the $n$-th roots
of unity. The natural number field for the QFT$_n$ measurement basis
is therefore the $n$-th cyclotomic field $\mathbb{Q}(\zeta_n)$.

This decomposition is the single-shot measurement primitive of the
Kyoto-Hiroshima 2025-09 W-state single-shot projective-measurement
work (per strategy synthesis F9 line). Substrate connection: for
$n = 5$ the natural field is the project's existing
\verb|QCyc5| substrate; for $n = 8$ it is contained in
\verb|QCyc16|; for $n = 40$ it is exactly the
\verb|QCyc40| FibonacciQutrit / T-gate-compiler natural number
field — the same cyclotomic substrate \textsection~\ref{sec:sk}'s
quantitative Solovay-Kitaev tight-$\varepsilon$ headline operates on.

\item \verb|endToEndFidelity| $= (1 + 3w^k)/4$ with $w = (4F-1)/3$, shown to
  satisfy the one-more-link swap recurrence, culminating in the capstone
  \verb|swapChain_fidelity_envelope|: for a $k$-swap Werner chain with
  per-link fidelity $F \in [1/4,1]$, the end-to-end fidelity provably lies in
  $[1/4,1]$, monotone in $F$ and antitone in chain length.
\end{itemize}
The envelope is \emph{model-independent} — a property of the fidelity (on which
the standard network simulators agree~\cite{ChungHajdusekVanMeter2025QNetSimXval})

\paragraph{Phase~6AB extensions (decay-inclusive, general Bell-diagonal, repeater breadth).}

real-parameter representation. (i)~\emph{Memory decoherence:}

$F(t)=F\,e^{-2t/\tau}+(1-e^{-2t/\tau})/4$, which cleanly \emph{multiplies the Werner

chain of memory-degraded links in $[1/4,1]$ given $F\in[1/4,1]$, $t\ge 0$, $\tau>0$ —
the realistic-network generalization. (ii)~\emph{General Bell-diagonal inputs:} the
Klein-4 ($\mathbb{Z}_2\times\mathbb{Z}_2$) convolution swap map
\verb|bellDiagSwapA|--\verb|D| with normalization
(\verb|bellDiagSwap_normalization|), nonnegativity, the $[0,1]$ target-fidelity

\paragraph{Phase~6AC extensions (operational metrics: key rate, multipartite, teleportation).}
Three operational network metrics are layered on the fidelity substrate, all
kernel-only. (i)~\emph{BB84 secret-key rate} (Shor--Preskill, Phys.\ Rev.\ Lett.\ \textbf{85}, 441 (2000)):
\verb|bb84KeyRate| $r(e)=1-2h_2(e)$ with
the bits-renormalized binary entropy $h_2(p)=\mathrm{binEntropy}(p)/\log 2$ (Mathlib's
\verb|binEntropy| is in nats), so $r(0)=1$. The positive-key crossover is
\emph{proven, not hardcoded}: \verb|bb84KeyRate_pos_iff_binEntropy_lt| states positivity as
$h_2(e)<1/2$, \verb|bb84KeyRate_strictAntiOn| gives strict decrease in the error rate,
\verb|bb84_crossover_exists| produces a genuine crossover $e^\ast\in(0,1/2)$ with $r(e^\ast)=0$
via the intermediate-value theorem, and \verb|bb84_positiveKey_fidelity_threshold| composes
this with the Phase-6AB end-to-end QBER (positive key iff $F_{e2e}>1-e^\ast$). The decimal
$e^\ast\approx 0.11$ is never asserted---it is the implicit root of $h_2(e)=1/2$.
(ii)~\emph{Multipartite GHZ-vs-W (Fortescue--Lo Thm 3.5):}
\verb|w3_beats_ghz_randomization_advantage| proves W$_3$'s randomization advantage over its
specified-pair single-copy bound ($2/3$) is strictly positive (reducing to the shipped
\verb|fortescueLoYield_gt_two_thirds|), whereas GHZ$_3$'s is the cited modeling input $0$;
\verb|w3_asymptotic_specified_lt_one| shows the W$_3$ asymptotic specified rate $H_2(1/3)<1$

probe-gated on the Haar--Pauli integral $\int_{S^2}(\langle\psi|\sigma_k|\psi\rangle)^2\,d\mu=1/3$,
which Mathlib at our pin lacks; rather than introduce an axiom we carry the value as the explicit

($f_{avg}=(2F+1)/3$ under the hypothesis) and the entanglement-utility threshold
\verb|teleport_beats_classical_iff| ($f_{avg}>2/3\Leftrightarrow F>1/2$, Massar--Popescu), composed
over the chain in \verb|teleport_useful_over_chain|. Kernel-only; \emph{zero} new project-local axioms.

\paragraph{Phase~6AD extensions (Haar integral discharged; Tier-1 anchors; general DEJMPS).}
The Horodecki proof's one analytic step is now \emph{proven}, removing the Phase-6AC
tracked hypothesis, and three reference-richness items are added, all kernel-only.
(i)~\emph{Haar--Pauli integral} (\verb|HaarPauli.lean|). In the Bloch picture (an explicit
two-component complex spinor and $2\times2$ Pauli matrix, no density matrices) we prove the
spinor identity $\langle\psi|\sigma_z|\psi\rangle=\cos\theta$ (\verb|pauliExpZ_blochKet|)
and the spherical integral $\int_{S^2}(\langle\psi|\sigma_z|\psi\rangle)^2\,d\mu=1/3$
(\verb|haarPauliZSqAverage_eq|, via the fundamental theorem of calculus on
$\int_0^\pi\cos^2\theta\,\sin\theta\,d\theta=2/3$ and the solid-angle normalization).
Consequently the Horodecki theorems become \emph{unconditional}:
\verb|teleportAvgFidelity_horodecki_unconditional| ($f_{avg}=(2F+1)/3$) and
\verb|teleport_beats_classical_iff_unconditional| hold with no hypothesis and no axiom.
(ii)~\emph{W1$'$ Tier-1 anchors} (\verb|Rate.lean|). The Calsamiglia--Lütkenhaus
linear-optics Bell-state-measurement bound (\verb|bsmSuccessProb_le_half_of_linearOptics|,
$\le 1/2$ vs the deterministic-BSM $1$) and the physics-only elementary-link rate
$\tau=L/(c\,p_{\mathrm{link}})$ (\verb|linkRate|), with the expected-attempts factor $1/p$
\emph{derived} as the geometric-distribution mean (\verb|geometric_expected_attempts|);
handshake-inclusive link times remain model-dependent (Tier-3) and are not bounded.
(iii)~\emph{General Bell-diagonal DEJMPS} (\verb|DEJMPSConvergence.lean|). The full Klein-4
map with normalization, nonnegativity, and pure-target fixed point; a monotone single-step
increase on the phase-flip-only sub-basin (\verb|dejmps_increase_phaseFlipOnly|); and a
verified non-monotonicity witness (\verb|dejmps_single_step_can_decrease|,
$(3/5,0,0,2/5)\mapsto 13/25<3/5$) showing that $\lambda_{00}>1/2$ does not guarantee a
single-step increase --- the full asymptotic basin rests on Macchiavello's non-monotone
argument and is cited, not formalized. A Python numeric mirror, three figures, and a
cross-validation gate (\verb|scripts/validate.py --check quantum_network|) accompany the
Lean substrate. Kernel-only; \emph{zero} new project-local axioms.

\paragraph{Phase~6AE (in progress): a general mixed-state / channel layer.}
Beyond the Bell-diagonal protocol class we have begun the general density-matrix layer

the \emph{linchpin} bridge \verb|traceNorm_posSemidef| ($\|A\|_1=\mathrm{tr}\,A$ for

\paragraph{Phase~6AF: the analytic core, discharged.} The metric, fidelity, and
data-processing layer that Phase~6AE deferred is now proven kernel-pure, bypassing the
machinery the 6AE note flagged as absent (no von Neumann / Ky Fan / polar decomposition needed):
\emph{(i)} the \textbf{trace-norm triangle} $\|A+B\|_1\le\|A\|_1+\|B\|_1$ --- both the Hermitian
case (\verb|traceNorm_hermitian_triangle|, via the positive-eigenvalue-sum decomposition
$\|H\|_1=2\sum\max(\lambda_i,0)-\mathrm{tr}\,H$ and a projection bound) and the \emph{general}
non-Hermitian case (\verb|traceNorm_triangle|, via the Hermitian dilation
$[\,0\ A;\ A^{\dagger}\ 0\,]$ over the doubled index $\iota\oplus\iota$, whose singular values
double $A$'s --- proven through the fact that the trace norm is a function of the characteristic
polynomial alone); hence $D(\rho,\sigma)$ is a genuine \textbf{metric}
(\verb|traceDist_triangle|) lying in $[0,1]$ (\verb|traceDist_mem_Icc|).
\emph{(ii)} the \textbf{operator modulus} $|A|=\sqrt{A^{\dagger}A}$ with $\|A\|_1=\mathrm{tr}\,|A|$.
\emph{(iii)} the \textbf{Uhlmann fidelity} $F(\rho,\sigma)=\mathrm{tr}\sqrt{\sqrt{\rho}\,\sigma\sqrt{\rho}}
=\|\sqrt{\sigma}\sqrt{\rho}\|_1$ (the trace-norm form proven equal to the literal Uhlmann
expression via $(\sqrt{\sigma}\sqrt{\rho})^{\dagger}(\sqrt{\sigma}\sqrt{\rho})=\sqrt{\rho}\,\sigma
\sqrt{\rho}$), with $F(\rho,\rho)=1$ and symmetry $F(\rho,\sigma)=F(\sigma,\rho)$.
\emph{(iv)} \textbf{CPTP data processing}: for a Kraus channel $\Phi(\rho)=\sum_kK_k\rho K_k^{\dagger}$
with $\sum_kK_k^{\dagger}K_k=\mathbf 1$, trace-norm contractivity $\|\Phi(A)\|_1\le\|A\|_1$ and the
\textbf{trace-distance contractivity} $D(\Phi\rho,\Phi\sigma)\le D(\rho,\sigma)$
(\verb|traceDist_krausMap_le|, via the positive/negative-part split, no dual norm).
\emph{(v)} \textbf{Choi positivity}: the Choi matrix of a Kraus channel is positive semidefinite
(\verb|choiMatrix_krausMap_posSemidef|, the channel--state-duality direction of Choi's theorem).
\emph{(vi)} the \textbf{diamond distance} $\mathrm{diamondDist}(\Phi_1,\Phi_2)=\sup_\rho
D((\Phi_1\otimes\mathrm{id})\rho,(\Phi_2\otimes\mathrm{id})\rho)=\tfrac12\|\Phi_1-\Phi_2\|_\diamond$:
after generalizing the trace-norm/CPTP layer to an arbitrary finite index (so it instantiates on
the doubled space $\mathrm{Fin}\,n\times\mathrm{Fin}\,n$ for free) and proving the stabilized
channel $\Phi\otimes\mathrm{id}$ is again CPTP (\verb|isKrausChannel_tensorKraus|, via the
Kronecker mixed-product identities), the supremum is well defined from boundedness alone
(\texttt{Real.sSup} needs no attainment, every term lying in $[0,1]$ as the stabilized outputs are
density operators); it is proven nonnegative, $\le 1$, symmetric, zero on the diagonal, and
\textbf{satisfying the triangle inequality} --- hence a genuine $[0,1]$-valued distinguishability
\emph{metric} on channels (\verb|diamondDist_nonneg|, \verb|diamondDist_le_one|,
\verb|diamondDist_comm|, \verb|diamondDist_self|, \verb|diamondDist_triangle|).
\emph{(vii)} the \textbf{fidelity range and the Fuchs--van de Graaf lower bound}: $F(\rho,\sigma)\le1$
(\verb|sqrtFidelity_le_one|, via an elementary double-Cauchy--Schwarz column assembly that bypasses
the absent Schatten-2 / matrix-H\"older layer), and the \textbf{Fuchs--van de Graaf lower bound}
$1-F\le D$ (\verb|one_sub_sqrtFidelity_le_traceDist|, a Powers--St\o rmer argument routed through a
matrix dual-norm bound $\mathrm{Re}\,\mathrm{tr}(HR)\le\|H\|_1$ for the sign operator $\mathrm{sgn}(S)$
and the positive/negative-part identity $\mathrm{tr}(\sqrt{\rho}\,P)\ge\mathrm{tr}\,P^2$ --- with
the non-commuting $|S|\le\sqrt{\rho}+\sqrt{\sigma}$ Loewner step \emph{removed}, not assumed).
\emph{(viii)} \textbf{diamond-distance attainment}: the supremum is \emph{achieved} by an optimal
input density operator (\verb|exists_diamondDist_eq|) --- so $\mathrm{diamondDist}$ is a genuine
maximum, not merely a least upper bound. The binding ingredient is \textbf{continuity of the
trace norm} (\verb|continuous_traceNorm|), obtained not from per-eigenvalue continuity (individual
eigenvalues are discontinuous at crossings) but from the elementary Lipschitz bound
$\bigl|\|A\|_1-\|B\|_1\bigr|\le\|A-B\|_1\le\sqrt{n}\,\|A-B\|_F$ (reverse triangle plus
Cauchy--Schwarz on the singular values, $\sum\sigma_i\le\sqrt n\,\sqrt{\sum\sigma_i^2}$ with
$\sum\sigma_i^2=\mathrm{tr}(A^{\dagger}A)=\|A\|_F^2$); the density-operator set is closed and
bounded, hence compact in the finite-dimensional Frobenius normed space, and the extreme value
theorem delivers the optimizer.
The remaining documented frontier (\verb|Phase6AF_Roadmap.md|), no \verb|sorry| or axiom: the
Fuchs--van de Graaf \textbf{upper} bound $D\le\sqrt{1-F^2}$ (every route needs Uhlmann's purification

characterization. Kernel-only; \emph{zero} new project-local axioms.

\section{Conclusion and outlook}
\label{sec:conclusion}

The D6 substrate sits at the convergence of four 2025--2026
fault-tolerant-quantum-computation results, all now populated and
kernel-verified: the Williamson--Yoder gauging-QEC overhead bound
(\textsection~\ref{sec:gaugingQEC}, Wave~6v.1), the Shor T-gate counts
for ECC-256 (\textsection~\ref{sec:shor}, Wave~6v.2), the APM-LDPC
hashing bound (\textsection~\ref{sec:apmldpc}, Wave~6v.5), and the
W-state QFT cyclotomic measurement primitive (\textsection~\ref{sec:wstate},
Wave~6v.6). The last of these is extended in
\textsection~\ref{sec:wstate:envelope} into a protocol-level
\emph{verified quantum-network substrate} (Phases~6AA--6AD): a
model-independent end-to-end fidelity envelope for entanglement-swap
chains, generalized to memory decoherence and general Bell-diagonal
inputs, together with the operational metrics network operators
care about --- the BB84 secret-key rate (with the positive-key
crossover proven, not hardcoded), the multipartite W$_3$-versus-GHZ$_3$
randomization advantage, and the Horodecki teleportation fidelity whose
sole analytic input (the Haar--Pauli integral $\int_{S^2}(\langle\psi|
\sigma_k|\psi\rangle)^2\,d\mu=1/3$) is itself discharged, leaving that
result unconditional and axiom-free. Every contribution is verified with
axioms $\subseteq\{$\texttt{propext}, \texttt{Classical.choice},
\texttt{Quot.sound}$\}$, with no \texttt{native\_decide} and no
project-local axioms; the Python numerics, figures, and a
cross-validation gate (\texttt{scripts/validate.py --check
quantum\_network}) mirror the Lean closed forms. The general density-matrix layer for arbitrary-state certification has since been built
(Phase~6AF, \textsection~\ref{sec:wstate:envelope}): the trace-distance metric, the operator
modulus, the Uhlmann fidelity, CPTP trace-distance contractivity (data processing), the
diamond distance $\tfrac12\|\Phi_1-\Phi_2\|_\diamond$ (a genuine metric, triangle inequality
included, with its supremum proven \emph{attained} by an optimal input state), the fidelity range
$F\le1$, and the Fuchs--van de Graaf \emph{lower} bound $1-F\le D$ are all proven kernel-pure. The
genuinely remaining extensions are the full asymptotic DEJMPS convergence basin (Macchiavello's
non-monotone argument) and two quantitative analytic items --- the Fuchs--van de Graaf \emph{upper}
bound $D\le\sqrt{1-F^2}$ (needs Uhlmann purification) and the diamond-norm Choi-SDP (Watrous)
characterization --- none required for the protocol-level results here. A bundle-level Stage~13 adversarial review has been run on
the quantum-network substrate; a consolidated full-bundle review across all
four headline results remains the final pre-submission gate.
