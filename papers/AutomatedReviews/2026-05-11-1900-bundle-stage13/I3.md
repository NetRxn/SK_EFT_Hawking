- **Gate:** LeanProofSubstance + NarrativeGrounding
- **Location:** `paper_draft.tex` lines 782–801 (`\begin{lstlisting}` typeclass-instance code block)

- **Why this matters:** the paper's claim "All five fields carry refutable Prop content" (lines 725–737 abstract; §5 line 750+) is **stronger than what the paper code-snippet illustrates** — the actual Lean carries the strengthened MGF / pushforward / centered-functional content from today's strengthening pass; the paper's code-snippet under-represents this. A reviewer comparing paper to source will conclude the paper has stale code.
- **Action:** refresh the §5 code block to the actual 4/5-tuple constructors; correspondingly add a short prose sentence noting that the post-strengthening Cramér predicates include MGF-normalization (`Continuous mgf ∧ mgf 0 ≤ 1`), the Sanov predicate adds an MGF reflexivity component, and the contraction predicate now requires the push-forward function to be continuous as well as I_X / I_Y.
