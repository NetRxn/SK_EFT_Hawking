### F5 — Cumulative-SNR integration-time chain — RECOMMENDED

- **Gate:** NarrativeGrounding / ProductionRunHealth
- **Location:** `papers/E2/paper_draft.tex:50-52`, `:280-283`, `:292-295`, `:351-352`
- **Observed:** Paper claims four times that "Bandwidth-cumulative SNR reaches unity in $\sim 1$ minute of integration at the Dean device" (and once "single dilution-fridge-day"). The named pipeline function `compute_graphene_spectrum.integration_time_cumulative` (`src/graphene/wkb_spectrum.py:198-203`) computes integration time using the per-Hz radiometer formula `t = 1/(Σ_i SNR_i² × Δf)` and returns **`3.716632270728482e-07` s** (verified by direct execution: `uv run python -c 'from src.graphene.wkb_spectrum import compute_graphene_spectrum; print(compute_graphene_spectrum("Dean_bilayer_nozzle").integration_time_s)'`). That is 3.7×10⁻⁷ s, not ~60 s — eight orders of magnitude apart.
- **Diagnosis:** The mismatch is not arithmetic error; it is two *different* SNR-to-integration-time prescriptions:
  - Pipeline `integration_time_cumulative`: per-Hz radiometer with white-Gaussian-noise assumption, no shape correlation, returns ~10⁻⁷ s for the Dean device. *This is the value the pipeline computes.*
  - Paper "~1 min": a Phase 5w shape-aware-correlation deep-research dossier estimate that accounts for Planckian-vs-white *shape* matching against finite measurement-bandwidth; this gives a much larger time because the shape correlation is much weaker than the per-bin SNR sum once thermal-floor systematics are included.
- **Expected:** A single body-text pointer in §V naming the shape-aware-correlation analysis (`Lit-Search/Phase-5w/...` reference or a sibling `compute_graphene_shape_correlation_t1min`-style helper) so the reader (and a future fresh-context reviewer) can trace which integration-time prescription the "~1 min" claim refers to. As written, "$\sim 1$ minute" reads as a pipeline output, but the pipeline's `integration_time_cumulative` returns 3.7×10⁻⁷ s.
- **Severity rationale:** RECOMMENDED, not BLOCKER, because both prescriptions are physically defensible and the paper's *direction* (achievable in dilution-fridge-day) is correct under either; what is missing is the cite that disambiguates which prescription the load-bearing minute estimate uses. Tier-4 letter has space for one parenthetical sentence + a `Lit-Search/Phase-5w/shape_aware_snr_correlation_*.md` reference. This is the canonical Stage 10 carry-forward item.
