C

Hybrid: Stacked with Slide-Out Detail

Same stacked layout as A, but clicking any axiom card or declaration row opens a slide-out panel from the right (like the KG detail panel) showing dep tree, statement, cross-links

Pros

  • Consistent with KG's existing detail panel pattern
  • Axioms prominent at top + deep drill-down on demand
  • Familiar to users of the KG tab already

Cons

  • Slide-out can feel cramped for tree views
  • Two panels competing for attention when open

My Recommendation: Option A

Stacked panels match the existing dashboard's visual language (parameters tab uses cards, lean tab uses tables). The axiom command panel at top gives you instant C2 visibility. The declaration browser below gives depth when you need it. Cross-links to the KG handle spatial exploration — no need to replicate that here. The vertical scroll concern is minor since the axiom panel is only 6 cards.
