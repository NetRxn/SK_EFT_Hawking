My Recommendation

Option A at full scale still feels clean — 2 axiom cards is compact, the declaration browser handles 58 modules well as a table, and vertical scroll is natural for this kind of data. Option B's sidebar works but gets long with 58 modules + all the module names. Option C is a good hybrid but the slide-out panel competes for horizontal space on narrower screens.

That said — with only 2 axioms, the command panel is lightweight enough that it could work in any layout. The declaration browser is where the real density lives.
